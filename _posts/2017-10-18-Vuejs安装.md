# {{ page.title }}

## 安装webpack
```
npm i webpack -g

```

## 安装vue-cli
```
npm i vue-cli -g
```

## 新建项目目录
```
mkdir 工程名称
```

## 根据模板创建项目
```
vue init webpack-simple 工程名称
注：工程名称不能是中文
```

## 生成项目所需的文件
```
cd 项目路径
npm install
```

## 运行项目
```
npm run dev

```
