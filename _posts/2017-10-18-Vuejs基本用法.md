## 注释
js注释部分注释与普通的js注释没有不同
```Javascript
export default {
  name: 'app',
  /*
  data () {
    return {
      msg: 'Welcome to Your Vue.js App',
      message: 'Hello,Message'
    }
  }*/
  data() {
    return {
      message:'bbb',
    }
  }
}

```
## 注册组件
```Javascript
import mycp from './component/mycp.vue'
export default {
  name: 'app',
  data() {
    return {
      message:'bbb',
    }
  },
  components: { mycp }
}
```

## 使用组件

```
<template>
```
